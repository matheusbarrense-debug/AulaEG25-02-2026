## 2. Requisitos Não Funcionais (RNF)

### RNF01 — Tempo de resposta
O sistema deve carregar telas principais em até **2 segundos**.

### RNF02 — Segurança
As senhas devem ser armazenadas usando hash seguro (ex.: bcrypt).

### RNF03 — Compatibilidade
O sistema deve funcionar nos navegadores Chrome, Edge e Firefox.

### RNF04 — Disponibilidade
O sistema deve garantir um uptime (tempo de atividade) de 99.9% (SLA).

### RNF05 — Escalabilidade
O sistema deve ser capaz de suportar até 1.000 usuários simultâneos sem degradação de performance.

### RNF06 — Responsividade
A interface do usuário (UI) deve ser adaptável para dispositivos móveis, tablets e desktops (Mobile First).

### RNF07 — Backup de Dados
O sistema deve realizar backups automáticos da base de dados a cada 24 horas.

### RNF08 — Internacionalização (i18n)
O sistema deve suportar múltiplos idiomas, iniciando por Português (PT-BR) e Inglês (EN-US).
